- [Node.js](https://nodejs.org/) (v14 ou superior)

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/username/repository.git

2. **Instale as dependências**
   npm install
   
3. **Rode o projeto**
   npm run dev
